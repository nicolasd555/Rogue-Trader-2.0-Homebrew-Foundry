# Rogue-Trader-2.0-Homebrew-Foundry
A foundry module used for playing Rogue Trader, with updates taken from Dark Heresy 2 implemented to streamline gameplay.

This system uses Easy LESS (an IDE add-on) to compile the css in any .less files into the main .css file. It is not needed to use the system, but is recommended if you intend to adjust the system's css.
